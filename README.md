# knst0/svm

My Stardew Valley mods. Each mod lives in its own folder with its own manifest, README and changelog, and is released independently.

## Mods

- [Genderbent Dialogue](/%5BCP%5D%20Genderbent%20Dialogue) — lore-friendly, data-only NPC genderbend for English and Russian. Flips the `Data/Characters` gender field and rewrites gendered dialogue, events and references to feminine; no textures. Toggle per character in Generic Mod Config Menu.

## Releases

Each mod is tagged and released on its own (`<slug>/v<version>`, e.g. `genderbent-dialogue/v1.0.0`). Pushing such a tag builds that mod's zip and publishes a GitHub release. See a mod's own README for install instructions.
